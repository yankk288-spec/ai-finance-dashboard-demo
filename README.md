# AI 财务单据解析与看板生成 Demo

这是一个可交互的前端 Demo，用于验证“客户上传财务数据 + 用自然语言描述报表需求 → AI 解析字段 → 理解业务意图 → 字段匹配校验 → 自动生成可解释财务看板 → 保存模板复用”的完整产品闭环。

## 当前支持场景

1. 费用分析
   - 部门费用、费用类型占比、每日费用趋势、异常大额支出、低置信度字段提醒。
2. P&L 利润损益分析
   - 收入、成本、毛利、期间费用、净利润、预算差异、部门利润贡献、异常科目波动。
3. P2P 异常付款检测
   - 重复付款、超 PO / 合同付款、无合同付款、提前付款、供应商账户变更、三单匹配。

## 核心流程

1. 选择业务场景，上传模拟财务文件并输入中文报表需求。
2. 展示模拟 OCR / 文档理解结果，包括单据或数据包类型、字段抽取、置信度与可计算字段。
3. 将自然语言需求拆解为主题、时间范围、指标、维度和推荐图表。
4. 校验报表组件所需字段、已解析字段、低置信度字段和缺失字段。
5. 生成对应业务场景看板，并支持查看数据来源、修改图表、保存模板。

## Mock AI 说明

以下能力均为本地模拟，不调用真实 OCR、LLM 或 BI 服务：

- 场景配置、字段模型、需求理解、字段匹配、看板数据：`src/data/scenarios.ts`
- 通用流程页面：`src/pages/`
- 通用可视化和来源弹窗：`src/components/`

所有演示数据均为虚构数据，不包含真实客户信息。

## 后续接入真实能力

- OCR / 文档理解：将上传文件发送到真实 OCR 服务，返回结构化字段、置信度和原文坐标。
- LLM 需求理解：把用户自然语言转换为指标、维度、过滤条件、图表类型和解释文本。
- 字段映射服务：沉淀客户字段到标准财务字段的映射规则，并支持人工确认回写。
- 分析规则引擎：为 P&L、P2P 等场景沉淀计算公式、异常规则和口径版本。
- BI / 低代码引擎：把字段模型转换为可配置图表组件、权限规则和模板资产。
- 模板复用：将单据模板、字段映射规则、看板模板保存到租户级或客户级资产库。
