# AI 财务单据解析与看板生成 Demo

这是一个可交互的前端 Demo，用于验证“客户上传财务单据 + 用自然语言描述看板需求 → AI 解析单据 → 理解看板需求 → 字段匹配校验 → 自动生成可解释财务看板 → 保存模板复用”的完整产品闭环。

## 核心流程

1. 上传模拟财务单据并输入中文看板需求。
2. 展示模拟 OCR 与文档理解结果，包括单据类型、字段抽取、置信度与可计算字段。
3. 将自然语言需求拆解为主题、时间范围、指标、维度和推荐图表。
4. 校验看板组件所需字段、已解析字段、低置信度字段和缺失字段。
5. 生成费用分析看板，并支持查看图表数据来源、修改图表、保存模板。

## Mock AI 说明

以下能力均为本地模拟，不调用真实 OCR、LLM 或 BI 服务：

- 单据类型识别与字段抽取：`src/data/mockDocuments.ts`
- 看板需求理解：`src/data/mockFieldMapping.ts`
- 字段匹配与缺失判断：`src/data/mockFieldMapping.ts`
- 看板指标与图表数据：`src/data/mockDashboard.ts`

所有演示数据均为虚构数据，不包含真实客户信息。

## 后续接入真实能力

- OCR / 文档理解：将上传文件发送到真实 OCR 服务，返回结构化字段、置信度和原文坐标。
- LLM 需求理解：把用户自然语言转换为指标、维度、过滤条件、图表类型和解释文本。
- 字段映射服务：沉淀客户字段到标准财务字段的映射规则，并支持人工确认回写。
- BI / 低代码引擎：把字段模型转换为可配置图表组件、权限规则和模板资产。
- 模板复用：将单据模板、字段映射规则、看板模板保存到租户级或客户级资产库。
